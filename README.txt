Это репозиторий для проекта. Здесь находятся диаграммы и word файл. Проект: “Программа с использованием GUI". Цель: создать программу, с индивидуально настраиваемым под каждого пользователя интерфейсом. Создатель выполняет работу по требованию пользователя и создает программу, соответствующую этим требованиям.
